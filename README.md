# keti-book-studio

도서 편집 및 단위 재계산 API 백엔드.

## 설치

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## 실행

```bash
uvicorn backend.main:app --reload
```

## 테스트

```bash
pytest
```
